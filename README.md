💱 Currency Converter

A responsive and user-friendly Currency Converter web application that allows users to convert currencies using real-time exchange rates. Built with HTML, CSS, and JavaScript, the application provides accurate currency conversion with an intuitive interface.

🚀 Live Demo

🔗 Live Website: https://sprint-currency-converter.onrender.com/

📂 GitHub Repository

🔗 Source Code: https://github.com/tanisha259/Currency_converter

---

✨ Features

- 🌍 Real-time currency conversion all over the world
- 💰 Support for multiple international currencies
- 🔄 Instant exchange rate calculations
- 📱 Fully responsive design
- ⚡ Fast and lightweight interface
- 🎨 Clean and modern UI
- 🛡️ Input validation for better user experience

🎯 How It Works

1. Enter the amount you want to convert.
2. Select the source currency.
3. Select the target currency.
4. Click the Convert button.
5. View the converted amount instantly.
